Overview

The aim of this project was to develop a system capable of identifying and localizing traffic objects such as cars, buses, and pedestrians in dynamic video environments. The methodology involved reading video files frame by frame and processing each frame using the pre-trained YOLOv3 model to detect objects. The OpenCV library was used for video processing and visualization, enabling the display of labeled bounding boxes around detected objects in real time. Additional utilities, such as imutils, facilitated video handling and frame rate monitoring, while NumPy supported necessary numerical computations. The integration of these tools ensured a streamlined pipeline for real-time object detection and visualization.

Result

The outcomes demonstrated the system's ability to efficiently detect and label traffic objects in videos with high accuracy and responsiveness, even in dynamic and complex environments. The model effectively tracked multiple objects simultaneously, providing robust real-time analysis.
