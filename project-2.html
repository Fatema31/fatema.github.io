A real-time traffic data collection and monitoring system is developed to facilitate the implementation of smart cities in developing countries like Bangladesh using a vision-based approach. In this work, various types of vehicle images such as cars, buses, trucks, motorcycles, and and three-wheelers are captured from different roads of Dhaka city. Using Roboflow, each image is annotated to identify and label the different types of vehicles. Dataset is split into 80:20 training-testing set ratio. YOLOv8 model is trained utilizing Darknet framework with iterative optimization to recognize that five distinct vehicle types. A track algorithm is used to assign unique IDs to detected objects and monitoring their movement across frames using Euclidean distance. Speed calculation is performed by analyzing the movement of vehicles between predefined lines on the video frame. Once each frame of a video is processed by the system, it writes the collected data to a csv file. The CSV file is initialized with the appropriate headers for each type of vehicles (such as ’Timestamp’, ’Car Up’, ’Car Down’, etc.). The system iteratively updates the counts of vehicles and average speed at regular intervals, typically every 10 seconds.



The CSV file can be served as a valuable resource for traffic monitoring, trend analysis, prediction, and decision making in various applications, including urban planning and transportation management. Furthermore, by using CSV format, the data can be easily imported into other data analysis tools and platforms for more in-depth analysis and visualization, enhancing the versatility and usability of the system’s output.
