Overview

A real-time traffic data collection and monitoring system is developed using a vision-based approach to facilitate the implementation of smart cities in developing countries like Bangladesh. In this work, various types of vehicle images, such as cars, buses, trucks, motorcycles, and three-wheelers, are captured from different roads of Dhaka city. Using Roboflow, each image is annotated to identify and label the different types of vehicles. The dataset is split into an 80:20 training-testing set ratio. The YOLOv8 model is trained with iterative optimization to recognize five distinct vehicle types. A tracking algorithm is used to assign unique IDs to detected objects and monitor their movement across frames using Euclidean distance. Speed calculation is performed by analyzing the movement of vehicles between predefined lines on the video frame. Once the system processes each frame of a video, the collected data is written in a CSV file. The CSV file is initialized with the appropriate headers for each type of vehicle (such as Timestamp, Car Up, Car Down, etc.). The system iteratively updates the counts of vehicles and average speed at regular intervals, typically every 10 seconds.

Method

One of the significant contributions was enabling YOLOv8 model in identifying the three wheelers like CNG (which is a common transport in Dhaka city but not widely used or recognized) effectively.



Result

The CSV file can be served as a valuable resource for traffic monitoring, trend analysis, prediction, and decision making in various applications, including urban planning and transportation management. Furthermore, by using CSV format, the data can be easily imported into other data analysis tools and platforms for more in-depth analysis and visualization, enhancing the versatility and usability of the system’s output.
